Piura-Hub

2021: This piece of group coursework was to design a website / app to help the local people of two towns called Lobitos and Piedritas. A demo version of the listings page, which I had control over can be accessed using the demo button in this page's header.

I do not have access to the server so nothing can be fetched from the database, however I have re-coded and populated the listings page so the responsive HTML and CSS I created can still be demonstrated (This is why the links have no function).

This page was an exercise in responsive design, and I am proud to have made a page which responds to different screen sizes and devices effectively. Below you can see the mobile version of the site.
